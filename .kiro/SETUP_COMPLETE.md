## 🚀 Nasıl Kullanılır?

### Yeni Bir Oturum Başladığında

**Adım 1:** Otomatik yükleme zaten çalışıyor ✅
- `steering/project-context.md` her oturumda otomatik yüklenir

**Adım 2:** Ana bağlamı yükle
```
"PROJECT_CONTEXT.md dosyasını oku"
```

**Adım 3:** İhtiyaca göre katalog dosyasını kullan
```
# Component arıyorsan
"COMPONENT_CATALOG.md'de sipariş component'lerini bul"

# Veritabanı işlemi yapacaksan
"DATABASE_GUIDE.md'de sipariş oluşturma query'sini göster"

# Hızlı başlangıç için
"QUICK_REFERENCE.md'ye göre yeni özellik ekleme workflow'unu takip et"
```

**Adım 4:** Sadece gerekli dosyayı oku
```
"components/orders/new-order-dialog.tsx dosyasını oku"
```

## 🐛 Sorun Giderme

### Problem: "Hala çok token harcıyorum"

**Çözüm:**
1. Aynı dosyaları tekrar tekrar okumayı bırak
2. Katalog dosyalarını kullan
3. Sadece gerekli dosyaları oku

### Problem: "Hangi dosyayı okuyacağımı bilmiyorum"

**Çözüm:**
1. `QUICK_REFERENCE.md` oku
2. "Senaryo Bazlı Dosya Seçimi" tablosuna bak

### Problem: "Bilgi güncel değil"

**Çözüm:**
1. İlgili dosyayı güncelle
2. "Son Güncelleme" tarihini değiştir
